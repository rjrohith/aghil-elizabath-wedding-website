# Elizabath & Aghil Wedding Website

## Preview correctly
1. Extract the ZIP first.
2. Keep `index.html`, `style.css`, `script.js` and the `assets` folder together.
3. Double-click `index.html`, or open the folder in VS Code and use the Live Server extension.

Do not open `index.html` from inside the ZIP archive; styling, images and animations may fail.

## RSVP
The included demo RSVP opens the guest's email application. For a production form, replace the JavaScript handler with Formspree, Tally or Google Forms.

## Hosting
This is a static site and can be hosted on GitHub Pages, Cloudflare Pages, Netlify or Vercel.

## Palette
